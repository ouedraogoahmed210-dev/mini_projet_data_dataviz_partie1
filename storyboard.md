1. **Issue** :Il y a une énorme inégalité dans le ciel d'Europe de l'Est : les vols réguliers sont très mal répartis entre les pays.

2. **Démonstration de l'issue** : Un graphique tout simple qui classe les pays par leur nombre d'aéroports actifs, où l'on voit que le premier pays est loin devant tous les autres.

3. **Idée / angle retenu** : Colorier uniquement les pays leader avec une couleur vive et laisser tous les autres en gris pour que la grosse différence saute aux yeux.
4. **Recommandation / conclusion** :Faire prendre conscience aux gens qu'il faut mieux équilibrer les transports pour éviter que certains pays se retrouvent complètement isolés
